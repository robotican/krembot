Installation

These instructions are compatible with Ubuntu 14.04 and 16.04.

Particle-cli

$ curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
$ sudo apt-get install -y nodejs
$ sudo npm install -g particle-cli
$ particle login

Particle-dev

Particle-dev is a Atom based ide for the photon.

Clone repository

$ cd ~
$  git clone https://github.com/robotican/krembot.git

Launch Atom and login with your account, see: https://docs.particle.io/guide/tools-and-features/dev/#logging-in

Open the ~/krembot/kermbot project folder:

File -> Add Project Folder -> ~/krembot/krembot
